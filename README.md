📌 Descrizione
Questo progetto simula il processo produttivo di un'azienda secondaria specializzata nella produzione di componenti elettronici. Il programma stima in modo realistico le risorse necessarie per produrre tre componenti fondamentali di uno smartphone:

🔋 Batteria
📱 Schermo
🖥️ Scheda Madre

Il modello genera quantità di produzione, tempi unitari e capacità giornaliera tramite valori casuali realistici, e calcola i tempi complessivi stimati per completare ogni lotto.

⚙️ Funzionalità

Generazione simulata delle quantità da produrre per ogni componente
Stima dei tempi di produzione unitari (in ore/minuti)
Calcolo della capacità produttiva giornaliera
Calcolo del numero di giorni necessari per completare la produzione
Stampa dei risultati a schermo in formato leggibile
Esportazione automatica dei risultati in formato CSV per ulteriori analisi


🛠️ Tecnologie utilizzate

Python 3.x
Librerie standard: random, csv
